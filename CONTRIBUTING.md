# Contributing to Raw Capture YT

First off, thank you for considering contributing to Raw Capture YT! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## Code of Conduct
This project and everyone participating in it is governed by our Code of Conduct. By participating, you are expected to uphold this code.

## How Can I Contribute?

### Reporting Bugs
Bugs are tracked as GitHub issues. When creating a bug report, please include:
- A clear, descriptive title.
- Steps to reproduce the bug.
- A description of the behavior you expected.
- Your OS and Browser version.

### Suggesting Enhancements
Enhancement suggestions are also tracked as GitHub issues. Provide:
- A descriptive title.
- A step-by-step description of the suggested enhancement.
- An explanation of why this would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the documentation is updated.
4. Issue that pull request!

## Style Guide
- Use descriptive variable names.
- Keep functions focused and small.
- Document all new features in the `docs/` folder.
